# 파이썬 함수 정리

* 리스트
  1. 모두 더할 때 : sum()
  2. 평균 값 구할 때 : np.mean()
  3. 자르기 : list[i:j]
  4. 뒤집기 : list[::-1]
  5. 중복 제거 : set()자료형 사용
  6. 맨 뒤에 추가 : append
  7. 맨 뒤에 값 반환 뒤 삭제 : pop
  8. pop(index) : 인덱스 위치의 값 삭제

* 문자열
  1. 공백으로 나누기 : str.split()
  2. lower, upper : 소문자, 대문자

* zip : 두 리스트를 튜플 형태로 변환

* set
  1. 교집합 : setA & setB
  2. 합집합 : setA | setB
  3. 차집합 : setA - setB
  4. 대칭차집합 : setA ^ setB
